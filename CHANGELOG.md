# Changelog — Selara Bot

> Версии нумеруются по дате: `YYYY-MM-DD`. Внутри каждой версии изменения разбиты по категориям.
> Для добавления новой записи: скопируй блок `## [YYYY-MM-DD]` в начало списка версий и заполни.

## [2026-06-02] — Система кланов

### Добавлено

- **Кланы** — новая социальная механика внутри чата.
  - `создать клан [название]` — создать клан (один клан на пользователя в чате).
  - `удалить клан` — удалить клан (только создатель; все участники исключаются автоматически).
  - `вступить в клан [id]` — вступить в существующий клан по числовому ID.
  - `выйти из клана` — покинуть клан (создатель не может покинуть — только удалить).
- **Профиль `/me`** — отображает строку `Клан: Название #id` если пользователь состоит в клане.
- **Кнопка «Мой клан»** — появляется под профилем `/me` если у пользователя есть клан; доступна любому зрителю профиля.
- **Карточка клана** — показывает название, ID, создателя, число участников и список состава.
- **Кнопка «Вступить»** — появляется на карточке клана для пользователей, не состоящих в нём.

### Техническое

- Новые таблицы БД: `clans`, `clan_members` (миграция `0045_add_clans`).
- Ограничение: уникальное имя клана в рамках одного чата, один клан на пользователя в чате.

## [2026-04-13] — Персоны

### Добавлено

- **Система персон** — каждому участнику чата можно назначить персону (метку).
  - Администраторы назначают метки через команды бота.
  - В настройках чата: `persona_enabled`, `persona_display_mode` (image_name и др.).
- Персона отображается рядом с именем пользователя в профиле и лидербордах.

### Техническое

- Новые колонки в `user_chat_activity`: `persona_label`, `persona_label_norm`, `persona_granted_by_user_id`, `persona_granted_at`.
- Новые колонки в `chat_settings`: `persona_enabled`, `persona_display_mode`.
- Миграция `0044_add_chat_personas`.

## [Ранее] — Базовые системы

### Добавлено (ретроспектива)

- **Активность и статистика** — `/me`, `/top`, `/rep`, `/lastseen`, `/active`.
- **Экономика** — монеты, фарм, магазин, рынок, аукцион, гача.
- **Отношения** — пары `/pair`, браки `/marry`, семья `/adopt`, `/pet`, `/family`.
- **Карма** — голосование `+` / `-`, отображение в профиле и лидерборде.
- **Модерация** — предупреждения `/warn`, преды `/pred`, баны `/ban`.
- **Достижения** — система ачивок с прогрессом и отображением в профиле.
- **Роли и права** — кастомные роли чата, иерархия, управление доступом к командам.
- **Игры** — мафия, карточные игры (ZlobCards), голосования.
- **Голосовой ввод** — STT интеграция (опционально).
- **Веб-панель** — FastAPI админка с авторизацией, просмотром чатов, пользователей, логов.
- **Триггеры и алиасы** — кастомные текстовые команды и RP-автоответы.
- **Архив сообщений** — хранение снапшотов сообщений с поиском.
