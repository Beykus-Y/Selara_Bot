# Словарь терминов чата (Glossary)

Словарь чата служит базой знаний для локального сленга, внутренних правил и определений.

1. **Использование словаря ассистентом**
   * При получении запроса, содержащего незнакомые термины, сленговые слова или аббревиатуры, ассистент должен сначала проверить их значение с помощью инструмента поиска.
   * **Поиск термина в словаре:** инструмент `lookup_glossary`
     * `term` (строка, обязательно): термин или слово для поиска (например: «рест», «варн», «топ»).

2. **Наполнение словаря**
   * Если в процессе общения или от администраторов бот узнает новое правило чата, расшифровку термина или локальное понятие, он должен записать это значение в словарь.
   * **Добавление термина в словарь:** инструмент `add_to_glossary`
     * `term` (строка, обязательно): термин (будет нормализован в нижний регистр).
     * `definition` (строка, обязательно): определение или описание термина.

*Важное правило:* Словарь является динамическим ресурсом. Все специфические для чата понятия бот должен изучать и дополнять в реальном времени исключительно через указанные инструменты.
