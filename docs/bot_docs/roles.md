# Роли и разграничение прав (Ранги)

В системе Selara пользователи разделены на несколько рангов (ролей), каждый из которых имеет свой уровень доступа к командам и функциям:

1. **Владелец (owner)**
   * Обладает абсолютными правами в чате.
   * Может выполнять любые действия, включая назначение других администраторов любого ранга.

2. **Старший администратор (senior_admin)**
   * Обладает высокими правами модерации.
   * Может использовать инструмент `set_rank` для назначения или изменения ролей других участников (за исключением понижения/изменения роли владельца).
   * Может выдавать и снимать любые дисциплинарные меры (варны, преды, баны, ресты).

3. **Младший администратор (junior_admin)**
   * Линейный модератор чата.
   * Может выдавать и снимать дисциплинарные меры: предупреждения (преды), выговоры (варны), баны и ресты.
   * **Ограничение:** не имеет доступа к инструменту `set_rank` (не может изменять роли пользователей).

4. **Участник (participant)**
   * Обычный пользователь чата.
   * Не имеет прав на использование модерационных инструментов AI-ассистента.

## Инструменты управления ролями
* **Изменить ранг/роль пользователя в боте:** инструмент `set_rank`
  * `target` (строка, обязательно): `@username`, образ или `user_id` пользователя.
  * `rank` (строка, обязательно): новая роль пользователя. Допустимые значения: `participant` | `junior_admin` | `senior_admin` | `owner`.

*Правило применения наказаний:* В чате отсутствует автоматический иммунитет администраторов к наказаниям. Дисциплинарные меры (варны, преды, баны) могут быть применены к любому участнику (включая младших/старших админов), если это явно затребовано владельцем (owner) или старшим администратором.
