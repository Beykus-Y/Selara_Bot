# Идеи для развития Selara Bot

---

## 💰 Экономика

### 1. Система кланов/гильдий
- Пользователи объединяются в кланы с общим банком
- Рейтинги кланов, клановые войны
- Совместные задания для бонусов клана
- Клановые достижения

### 2. Квесты и миссии
- Ежедневные/еженедельные квесты с наградами
- Цепочки квестов (storyline)
- Квесты на взаимодействие с другими участниками
- Сезонные квесты с уникальными наградами

### 3. Система уровней и XP
- Опыт за активность в чате, игры, экономику
- Уровни открывают новые возможности/команды
- Визуальные бейджи за уровни
- Таблица лидеров по уровням

### 4. Банковская система
- Вклады с процентами
- Кредиты между пользователями
- Инвестиции в "акции" других пользователей
- Страхование от потерь

### 5. Предсказания и ставки
- Пользователи создают предсказания (да/нет)
- Остальные ставят монеты на исход
- Автоматическое разрешение по результату
- Ставки на события в чате

### 6. Случайные события
- "Налог" — случайный сбор монет
- "Находка" — бонус случайному участнику
- "Праздник" — удвоенные награды на время
- Стихийные бедствия, влияющие на фермы

### 7. Расширенная торговля
- Бартер между пользователями (обмен предметами)
- Аукционы с таймером и авто-ставками
- Пакетные предложения
- Система отзывов о продавцах

---

## 🎮 Игры

### 8. Новые игры
- **Камень-ножницы-бумага** — PvP с ставками
- **Рулетка** — классическая с монетами
- **Слот-машина** — однорукий бандит
- **Блэкджек** — карточная игра
- **Покер** — Texas Hold'em для компании
- **Нарды/Шахматы** — через inline-кнопки

### 9. RPG-приключения
- Пошаговые подземелья для группы
- Выбор действий влияет на исход
- Лут и награды за прохождение
- Классы персонажей (воин, маг, вор)

### 10. Турнирная система
- Автоматические турниры по играм
- Скобки (brackets) и рейтинги
- Призовой фонд из взносов
- Сезонные чемпионаты

### 11. Коллекционные карточки (расширение гачи)
- Обмен карточками между пользователями
- Редкие/фольгированные версии
- Коллекции с бонусами за завершение
- Карточки на основе мемов чата

## 👥 Социальные фичи

### 12. Виртуальные питомцы
- Кормление, уход, игры с питомцем
- Питомцы растут и эволюционируют
- Питомцы дают бонусы к экономике
- Соревнования питомцев

### 13. Система репутации 2.0
- Категории репутации (юмор, помощь, активность)
- Визуальные значки репутации
- История репутации
- Репутация влияет на бонусы

### 14. Газета/дайджест чата
- Ежедневная сводка активности
- Топ участников дня
- Лучшие мемы/сообщения
- Экономические итоги

### 15. Календарь событий
- Планирование игровых сессий
- Напоминания о событиях
- RSVP (подтверждение участия)
- Интеграция с играми бота

### 16. Реферальная система
- Награды за приглашение друзей
- Бонусы за активность рефералов
- Дерево рефералов
- Специальные награды за milestones

### 17. Профили пользователей 2.0
- Кастомные аватары
- Био/о себе с форматированием
- Витрина достижений
- Статистика в красивом формате

## 📋 Мелкие улучшения

### 34. Улучшения существующих фич
- **Достижения с наградами** — привязать монеты/предметы к достижениям
- **Роль game_id текстом** — `роль <game_id>` через текст (сейчас только slash)
- **chat_write_locked блокирует команды** — middleware для блокировки
- **Web показывает статус блокировки** — chat_write_locked/antiraid в UI
- **Система потребностей персонажа** — деградация без ухода
- **Расширенные настройки фермы** — авто-полив, авто-сбор
- **Комплектные бонусы** — бонус за набор предметов
- **Сезонные товары** — лимитированные предметы

### 35. UX улучшения
- Подсказки команд при вводе
- Интерактивные туториалы для новых пользователей
- Уведомления о кулдаунах
- Быстрые действия через inline-кнопки

### 36. Производительность
- Оптимизация кэширования
- Фоновая обработка тяжёлых операций
- Пагинация для больших списков
- Lazy loading в web-панели
